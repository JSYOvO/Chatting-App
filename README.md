# Chatting WEB 🚀

<br/>

## 🔎 프로젝트 소개

다양한 기술스택을 사용한 다양한 버젼의 CHATTING WEB을 빌드합니다.

<br/>

## 🧱 기술 스택

React + TypeScript
<br/>
Socket.io
<br/>
Node Express
<br/>
MongoDB + Mongoose
<br/>

<br/>

## 💻 [데모](https://chatting-web-jsyovo.netlify.app/)
